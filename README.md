# Dreamweaver — Calm and Relax 🌿

A 20-minute guided sleep hypnosis experience set in a tranquil rainforest.

## Features
- 🌸 Animated lotus flower with breathing animation
- 🎋 Deep forest green colour scheme
- 🌧️ Rainforest ambient soundscape
- 💚 39-line guided meditation script

## How to Use
Open `index.html` in your browser and click **Enter the Forest**.

## Audio Files Needed
Place the following in the `/audio/` folder:
- `rainforest_ambient.mp3` — looping rainforest sounds
- `rainforest_line_00.mp3` through `rainforest_line_38.mp3` — voice lines
